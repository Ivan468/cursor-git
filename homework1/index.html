GIT commands

Команди гіта


  • git config --global user.name blabla - вводить ім'я того хто буде міняти файли
  • git config --global user.email blabal@bla емайл користувача
  • git init - іціціювати гіт в папці локальній
  • git status - перевірити статус гіта
  • git add - додати файли до гіта щоб в майбутньому закомітити
  • git commit - закомітити файли
  • git log - показати коміти (історію редагувань)
  • git checkout - змінити гілку (на головну наприклад)
  • git branch - створити гілку
  • git push - вигрузити локальні зміни на віддалений віддалений сервак (гітХаб)
  • git pull - загрузити віддалені зміни із сервера на ПК
  • git merge - з'єднати гілки або гілку (з головною наприклад)
  • git clone - зклонувати із віддаленого сервака собі на ПК файли

.gitignore файл - це файл який можна створити в Гіті щоб ігнорувати деякі інші файли які не потрібно додавати в комміти
